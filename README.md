# zeek_helpers
Scripts to summarize NSM logs.

These scripts are for summerizing small zeek logs.

If you are looking for scripts to aide in summerizing zeek logs, I suggest you do not use these. Instead, define your purpose and desired outcome, familiarize yourself with Zeek and the concepts behind Network Security Monitoring, and use a coding agent to quickly build your scripts in your desired language. 

These scripts only parse text log files.
